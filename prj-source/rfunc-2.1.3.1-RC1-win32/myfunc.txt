Включение новых функций в rfunc.
1. Создаем файл myfunc.c, куда включаем все дополнительные функции.
2. В файле сборки makefile.bc корректируем макроопределения IBASE, BCDIR, SYSDIR, IBVERSION в соответствии
с используемыми версиями Interbase и компилятора.
3. Добавляем в определение OBJS myfunc.obj (или myfunc.o, взависимости от платформы).
4. Собираем библиотеку командой
	make -DREXT=myfunc -f makefile.bc all
5. Инсталлируем библиотеку командой
	make -f makefile.bc install
6. Объявляем функции в базе данных.

Создание собственной библиотеки на базе rfunc.
1. Создаем файл myfunc.c (просто копируем файл rfunc.c), куда включаем все дополнительные функции.
2. В файле сборки makefile.bc корректируем макроопределение TARGETNAME - прописываем название результирующей библиотеки.
TARGETNAME=	myfunc
3. Собираем библиотеку
	make -f makefile.bc all
4. Инсталлируем библиотеку
	make -f makefile.bc install
5. Корректируем объявление существующих функций в rfunc.sql и объявляем описание для новых функции.
	Изменяем имя библитеки на myfunc.
6. Объявляем функции в базе данных.

Создание собственной библиотеки с использованием файла сборки makefile.bc.
1. Создаем файл myfunc.c, куда включаем все функции.
2. В файле сборки makefile.bc корректируем макроопределения IBASE, BCDIR, SYSDIR, IBVERSION в соответствии
с используемыми версиями Interbase и компилятора.
3. Собираем библиотеку командой
	make -f makefile.bc myfunc.dll
4. Изменяем параметр TARGETNAME на myfunc. Инсталлируем библиотеку командой
	make -f makefile.bc install
6. Объявляем функции в базе данных.
